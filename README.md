# AI News Hub

A Full Stack AI News Dashboard built using:

- React.js
- Django REST Framework
- GNews API

## Features

- Top 5 AI News Articles
- Duplicate News Filtering
- News Images
- Last Updated Timestamp
- Hourly Refresh Support
- Secure Backend API

## Tech Stack

Frontend:
- React
- CSS

Backend:
- Django
- Django REST Framework

API:
- GNews
